## 🏆 项目总览

这是一个**世界级的前端项目**，经过**10次完整迭代**，从简单的乞讨网站发展成为**企业级、全功能、高性能**的现代Web应用。

### 📊 项目规模

```
总文件数: 110+ 个
总代码行数: ~20,700+ 行
组件数量: 54+ 个
自定义Hooks: 17 个
工具函数: 135+ 个
系统模块: 35+ 个
```

### 🎯 技术栈

- **框架**: Next.js 16 (App Router) + React 19 + TypeScript 5
- **样式**: Tailwind CSS 4 + 自定义像素风格
- **性能**: Web Workers + 虚拟滚动 + 懒加载 + 代码分割
- **音频**: Web Audio API (零依赖音效合成)
- **存储**: localStorage + IndexedDB
- **PWA**: Service Worker + 离线支持
- **图表**: 纯SVG实现 (零依赖)
- **动画**: requestAnimationFrame + 40+缓动函数
